WORKFLOW
========

1. Setup:
   cp .env.example .env
   # OpenAI API Key eintragen
   npm install

2. Layer generieren:
   npm run generate
   # Erzeugt alle Belag-Layer in generated/raw/ und generated/clean/

3. Manifest bauen:
   npm run manifest
   # Erzeugt generated/layers.js und generated/layers.json

4. Oder alles auf einmal:
   npm run build

5. Preview erzeugen:
   npm run preview
   # Erzeugt generated/pizza-preview.html
   # Oeffne die Datei im Browser


INTEGRATION IN DEINE WEBSITE
==============================

Option A — Script-Tag:
  <script src="generated/layers.js"></script>
  <script>
    console.log(PIZZA_LAYERS.layers);
  </script>

Option B — ES Import:
  import { PIZZA_LAYERS } from './generated/layers.js';

Option C — JSON Fetch:
  fetch('generated/layers.json')
    .then(r => r.json())
    .then(data => console.log(data.layers));


LAYER-SCHICHTUNG IM KONFIGURATOR
==================================

Von unten nach oben:
  1. Teig-Bild (dein base-tomate.png etc.)
  2. Belag-Layer (die generierten PNGs) ← z-index: 5
  3. Kaese-Overlay (dein cheese-tomate.png) ← z-index: 10

Die Belag-Layer liegen ZWISCHEN Teig und Kaese,
dadurch sieht es aus als waeren die Belaege unter dem geschmolzenen Kaese.
